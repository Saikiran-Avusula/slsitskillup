# SLSIT Skillup - Portfolio Description

## 🎯 Project Overview
A modern, fully responsive educational platform website for SLSIT Skillup - a professional skill development institute offering technology courses and career training programs.

## 🛠️ Tech Stack
- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS + Custom CSS
- **Animations**: Framer Motion
- **Routing**: React Router v6
- **Icons**: Lucide React
- **Backend**: Node.js + Express (Email service)

## ✨ Key Features
- **Responsive Design**: Fully optimized for mobile (320px), tablet (768px), and desktop (1920px+)
- **Interactive Animations**: Smooth page transitions, hover effects, and scroll animations
- **Course Management**: Dynamic course catalog with detailed pages and enrollment system
- **Contact System**: Integrated contact forms with email service
- **Modern UI**: Glass morphism effects, gradient animations, and particle backgrounds
- **CRT Program**: Dedicated Campus Recruitment Training section
- **Chat Widget**: Interactive support widget for user engagement

## 🎨 Design Highlights
- Story-driven UI with chapter-based content flow
- Professional gradient color schemes (Blue to Purple)
- Glass morphism cards with backdrop blur effects
- Animated particle background system
- Responsive navigation with mobile hamburger menu
- Custom scrollbar and loading animations

## 📱 Responsive Features
- Mobile-first approach with breakpoint optimization
- Touch-friendly buttons (44px+ touch targets)
